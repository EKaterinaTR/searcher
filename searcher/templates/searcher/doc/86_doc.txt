<p><b>Происхождение птиц</b> — <a href="/wiki/%D0%AD%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F" title="Эволюция">эволюционный</a> процесс, в результате которого образовалась группа <a href="/wiki/%D0%9F%D1%82%D0%B8%D1%86%D1%8B" title="Птицы">птиц</a> (Aves). Согласно современным представлениям, <a href="/wiki/%D0%9F%D1%82%D0%B8%D1%86%D1%8B" title="Птицы">птицы</a> являются специализированными <a href="/wiki/%D0%A2%D0%B5%D1%80%D0%BE%D0%BF%D0%BE%D0%B4%D1%8B" title="Тероподы">тероподовыми</a> <a href="/wiki/%D0%94%D0%B8%D0%BD%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D1%8B" title="Динозавры">динозаврами</a> из <a href="/wiki/%D0%9A%D0%BB%D0%B0%D0%B4%D0%B0" title="Клада">клады</a> <a href="/wiki/%D0%9C%D0%B0%D0%BD%D0%B8%D1%80%D0%B0%D0%BF%D1%82%D0%BE%D1%80%D1%8B" title="Манирапторы">манирапторов</a>, близкими к другим <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D1%91%D0%BD%D0%BD%D1%8B%D0%B5_%D0%B4%D0%B8%D0%BD%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D1%8B" title="Оперённые динозавры">оперённым динозаврам</a>, таким как <a href="/wiki/%D0%94%D1%80%D0%BE%D0%BC%D0%B5%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D0%B8%D0%B4%D1%8B" title="Дромеозавриды">дромеозавриды</a> и <a href="/wiki/%D0%A2%D1%80%D0%BE%D0%BE%D0%B4%D0%BE%D0%BD%D1%82%D0%B8%D0%B4%D1%8B" title="Троодонтиды">троодонтиды</a>.
Традиционно самым древним из известных родов птиц считается <a href="/wiki/%D0%90%D1%80%D1%85%D0%B5%D0%BE%D0%BF%D1%82%D0%B5%D1%80%D0%B8%D0%BA%D1%81" title="Археоптерикс">археоптерикс</a> (<i>Archaeopteryx</i>), живший в <a href="/wiki/%D0%9F%D0%BE%D0%B7%D0%B4%D0%BD%D1%8F%D1%8F_%D1%8E%D1%80%D0%B0" title="Поздняя юра">позднеюрскую эпоху</a>.
</p><p>Единичные авторы придерживаются альтернативной версии, согласно которой, древнейшей известной птицей или близкой к ним формой является <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%BE%D0%B0%D0%B2%D0%B8%D1%81" title="Протоавис">протоавис</a> из позднего <a href="/wiki/%D0%A2%D1%80%D0%B8%D0%B0%D1%81%D0%BE%D0%B2%D1%8B%D0%B9_%D0%BF%D0%B5%D1%80%D0%B8%D0%BE%D0%B4" title="Триасовый период">триаса</a><sup id="cite_ref-Kurochkin_2015_1-0" class="reference"><a href="#cite_note-Kurochkin_2015-1">&#91;1&#93;</a></sup>. По этой версии, и птицы, и тероподы произошли от одной из более ранних групп <a href="/wiki/%D0%90%D1%80%D1%85%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D1%8B" title="Архозавры">архозавров</a>. Большинство специалистов считают протоависа <a href="/wiki/%D0%A5%D0%B8%D0%BC%D0%B5%D1%80%D0%B0_(%D0%BF%D0%B0%D0%BB%D0%B5%D0%BE%D0%BD%D1%82%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Химера (палеонтология)">химерой</a> и не принимают во внимание<sup id="cite_ref-Zhou_2004_2-0" class="reference"><a href="#cite_note-Zhou_2004-2">&#91;2&#93;</a></sup><sup id="cite_ref-Mayr_2017_introduction_3-0" class="reference"><a href="#cite_note-Mayr_2017_introduction-3">&#91;3&#93;</a></sup>.
</p>

<h2><span id=".D0.9A.D0.BB.D0.B0.D1.81.D1.81.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.B8.D0.B5_.D0.B3.D0.B8.D0.BF.D0.BE.D1.82.D0.B5.D0.B7.D1.8B"></span><span class="mw-headline" id="Классические_гипотезы">Классические гипотезы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Редактировать раздел «Классические гипотезы»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;action=edit&amp;section=1" title="Редактировать раздел «Классические гипотезы»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id=".C2.AB.D0.A1_.D0.B7.D0.B5.D0.BC.D0.BB.D0.B8_.D0.B2.D0.B2.D0.B5.D1.80.D1.85.C2.BB_.28ground_up.3B_cursorial.29"></span><span class="mw-headline" id="«С_земли_вверх»_(ground_up;_cursorial)">«С земли вверх» (ground up; cursorial)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Редактировать раздел ««С земли вверх» (ground up; cursorial)»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;action=edit&amp;section=2" title="Редактировать раздел ««С земли вверх» (ground up; cursorial)»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Гипотезы из группы «с земли вверх» выводят птиц непосредственно из наземных животных и исторически связаны с динозавровой версией происхождения птиц. Эта версия первоначально была выдвинута <a href="/wiki/%D0%A5%D0%B0%D0%BA%D1%81%D0%BB%D0%B8,_%D0%A2%D0%BE%D0%BC%D0%B0%D1%81_%D0%93%D0%B5%D0%BD%D1%80%D0%B8" class="mw-redirect" title="Хаксли, Томас Генри">Томасом Генри Хаксли</a> в XIX веке. В 70-х годах XX века её возродил <a href="/wiki/%D0%9E%D1%81%D1%82%D1%80%D0%BE%D0%BC,_%D0%94%D0%B6%D0%BE%D0%BD" title="Остром, Джон">Джон Остром</a>. Основываясь на сравнении пересмотренной им <a href="/wiki/%D0%9E%D1%81%D1%82%D0%B5%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F" title="Остеология">остеологии</a> археоптерикса и <a href="/wiki/%D0%A2%D0%B5%D1%80%D0%BE%D0%BF%D0%BE%D0%B4%D1%8B" title="Тероподы">тероподных динозавров</a> (в особенности описанного им <a href="/wiki/%D0%94%D0%B5%D0%B9%D0%BD%D0%BE%D0%BD%D0%B8%D1%85" title="Дейноних">дейнониха</a>), Остром заключил, что ближайшими родственниками археоптерикса являются <a href="/wiki/%D0%A6%D0%B5%D0%BB%D1%83%D1%80%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D1%8B" title="Целурозавры">целурозавры</a> из семейства <a href="/wiki/%D0%94%D1%80%D0%BE%D0%BC%D0%B5%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D0%B8%D0%B4%D1%8B" title="Дромеозавриды">дромеозаврид</a>.
</p><p>Впоследствии выводы Острома были подтверждены <a href="/wiki/%D0%9A%D0%BB%D0%B0%D0%B4%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0" title="Кладистика">кладистическими исследованиями</a>, обнаружением экземпляров целурозавров с отпечатками перьев, сравнением микроструктуры яичной скорлупы и репродуктивной биологии целурозавров и птиц<sup id="cite_ref-Gauthier_1986_4-0" class="reference"><a href="#cite_note-Gauthier_1986-4">&#91;4&#93;</a></sup><sup id="cite_ref-toeeb_5-0" class="reference"><a href="#cite_note-toeeb-5">&#91;5&#93;</a></sup><sup id="cite_ref-6" class="reference"><a href="#cite_note-6">&#91;6&#93;</a></sup>. Сегодня тероподная гипотеза принимается большинством палеонтологов.
</p><p>Современная <a href="/wiki/%D0%A4%D0%B8%D0%BB%D0%BE%D0%B3%D0%B5%D0%BD%D0%B8%D1%8F" class="mw-redirect" title="Филогения">филогения</a> птиц и наиболее близких к ним групп тероподов выглядит следующим образом:
</p>
<blockquote>
<ul><li><a href="/wiki/Tetanurae" class="mw-redirect" title="Tetanurae">Tetanurae</a>
<ul><li><a href="/wiki/Coelurosauria" class="mw-redirect" title="Coelurosauria">Coelurosauria</a> — Целурозавры
<ul><li><a href="/wiki/Maniraptoriformes" class="mw-redirect" title="Maniraptoriformes">Maniraptoriformes</a>
<ul><li><a href="/wiki/Maniraptora" class="mw-redirect" title="Maniraptora">Maniraptora</a>
<ul><li><a href="/wiki/Eumaniraptora" class="mw-redirect" title="Eumaniraptora">Eumaniraptora</a>
<ul><li><a href="/wiki/Deinonychosauria" class="mw-redirect" title="Deinonychosauria">Deinonychosauria</a> (<a href="/wiki/Dromaeosauridae" class="mw-redirect" title="Dromaeosauridae">Dromaeosauridae</a> + <a href="/wiki/Troodontidae" class="mw-redirect" title="Troodontidae">Troodontidae</a>)</li>
<li><a href="/wiki/Aves" class="mw-redirect" title="Aves">Aves</a>&#160;— Птицы
<ul><li><a href="/wiki/Archaeopterygiformes" class="mw-redirect" title="Archaeopterygiformes">Archaeopterygiformes</a></li>
<li><a href="/wiki/Pygostylia" title="Pygostylia">Pygostylia</a>
<ul><li><a href="/wiki/Neornithes" class="mw-redirect" title="Neornithes">Neornithes</a></li></ul></li></ul></li></ul></li></ul></li></ul></li></ul></li></ul></li></ul>
</blockquote>
<p>Поскольку птицы оказались близкими родственниками двуногих наземных хищников, возник вопрос, как у таких животных возник полёт. Разные авторы выдвинули несколько гипотез, которые объединяют в группу «с земли вверх»<sup id="cite_ref-Wang_2017_7-0" class="reference"><a href="#cite_note-Wang_2017-7">&#91;7&#93;</a></sup><sup id="cite_ref-Kurochkin_2008_8-0" class="reference"><a href="#cite_note-Kurochkin_2008-8">&#91;8&#93;</a></sup>. Впервые такую идею высказал<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup> <a href="/wiki/%D0%A3%D0%B8%D0%BB%D0%BB%D0%B8%D1%81%D1%82%D0%BE%D0%BD,_%D0%A1%D1%8D%D0%BC%D1%8E%D1%8D%D0%BB%D1%8C" title="Уиллистон, Сэмюэль">Сэмюэль Уиллистон</a> в 1879 году<sup id="cite_ref-Ostrom_1974_10-0" class="reference"><a href="#cite_note-Ostrom_1974-10">&#91;10&#93;</a></sup>. Остром в 1976 году предположил, что превращение передних конечностей в крылья было связано с их использованием для ловли <a href="/wiki/%D0%9D%D0%B0%D1%81%D0%B5%D0%BA%D0%BE%D0%BC%D1%8B%D0%B5" title="Насекомые">насекомых</a>. Эта гипотеза неоднократно критиковалась, и в 1997 году Остром от неё отказался<sup id="cite_ref-Kurochkin_2008_8-1" class="reference"><a href="#cite_note-Kurochkin_2008-8">&#91;8&#93;</a></sup>.
</p><p>Вариант гипотезы «с земли вверх»&#160;— <span data-interwiki-lang="en" data-interwiki-article="Wing-assisted incline running"><a href="/w/index.php?title=%D0%91%D0%B5%D0%B3_%D0%BF%D0%BE_%D1%81%D0%BA%D0%BB%D0%BE%D0%BD%D1%83_%D1%81_%D0%BF%D0%BE%D0%BC%D0%BE%D1%89%D1%8C%D1%8E_%D0%BA%D1%80%D1%8B%D0%BB%D1%8C%D0%B5%D0%B2&amp;action=edit&amp;redlink=1" class="new" title="Бег по склону с помощью крыльев (страница отсутствует)">бег по склону с помощью крыльев</a></span><span class="noprint" style="white-space: nowrap; font-size: 85%;"> (<a href="https://en.wikipedia.org/wiki/Wing-assisted_incline_running" class="extiw" title="en:Wing-assisted incline running"><span title="Wing-assisted incline running — версия статьи «Бег по склону с помощью крыльев» на английском языке">англ.</span></a>)</span>&#160;— предложил<sup id="cite_ref-Dial_2003_11-0" class="reference"><a href="#cite_note-Dial_2003-11">&#91;11&#93;</a></sup> в 2003 году Кеннет Диал на основе наблюдения птенцов <a href="/wiki/%D0%9A%D0%B5%D0%BA%D0%BB%D0%B8%D0%BA" class="mw-redirect" title="Кеклик">кекликов</a>, взбиравшихся по наклонным поверхностям (вплоть до почти вертикальных), помогая себе взмахами крыльев<sup id="cite_ref-Kurochkin_2008_8-2" class="reference"><a href="#cite_note-Kurochkin_2008-8">&#91;8&#93;</a></sup><sup id="cite_ref-Wang_2017_7-1" class="reference"><a href="#cite_note-Wang_2017-7">&#91;7&#93;</a></sup>.
</p>
<h3><span id=".C2.AB.D0.A1_.D0.B4.D0.B5.D1.80.D0.B5.D0.B2.D1.8C.D0.B5.D0.B2_.D0.B2.D0.BD.D0.B8.D0.B7.C2.BB_.28arboreal.3B_trees_down.29"></span><span class="mw-headline" id="«С_деревьев_вниз»_(arboreal;_trees_down)">«С деревьев вниз» (arboreal; trees down)</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Редактировать раздел ««С деревьев вниз» (arboreal; trees down)»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;action=edit&amp;section=3" title="Редактировать раздел ««С деревьев вниз» (arboreal; trees down)»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Эту гипотезу предложил<sup id="cite_ref-Marsh_1880_12-0" class="reference"><a href="#cite_note-Marsh_1880-12">&#91;12&#93;</a></sup> в 1880 году <a href="/wiki/%D0%9C%D0%B0%D1%80%D1%88,_%D0%9E%D1%82%D0%BD%D0%B8%D0%B5%D0%BB_%D0%A7%D0%B0%D1%80%D0%BB%D0%B7" title="Марш, Отниел Чарлз">Отниел Чарлз Марш</a><sup id="cite_ref-Ostrom_1974_10-1" class="reference"><a href="#cite_note-Ostrom_1974-10">&#91;10&#93;</a></sup>. Её развивали сторонники происхождения птиц от древесных рептилий. Так, в начале <a href="/wiki/XX_%D0%B2%D0%B5%D0%BA" title="XX век">XX века</a> <a href="https://en.wikipedia.org/wiki/Gerhard_Heilmann" class="extiw" title="en:Gerhard Heilmann"><span title="Gerhard Heilmann — версия статьи «Хайльман, Герхард» на английском языке">Герхард Хайльман</span></a><span class="noprint" style="font-style:normal; font-weight:normal;">&#160;<span style="display:inline-block; font-size:80%; position:relative;"><span style="font-size:117.6%; margin-left:-0.43em; position:relative; top:0.28em;">&#160;<small class="ref-info" style="cursor:help;" title="на английском языке">(англ.)</small></span><span class="link-ru metadata" style="margin-left:-0.43em; position:absolute; left:0; z-index:1; margin-top:-0.35em;"><span style="visibility:hidden; margin-right:0.099em;">&#160;(</span><span data-interwiki-lang="en" data-interwiki-article="Gerhard Heilmann"><a href="/w/index.php?title=%D0%A5%D0%B0%D0%B9%D0%BB%D1%8C%D0%BC%D0%B0%D0%BD,_%D0%93%D0%B5%D1%80%D1%85%D0%B0%D1%80%D0%B4&amp;action=edit&amp;redlink=1" class="new" title="Хайльман, Герхард (страница отсутствует)">рус.</a></span></span></span></span>, основываясь на детальном обзоре биологии птиц (включая археоптерикса) и <a href="/wiki/%D0%A0%D0%B5%D0%BF%D1%82%D0%B8%D0%BB%D0%B8%D0%B8" class="mw-redirect" title="Рептилии">рептилий</a>, сделал вывод, что самыми близкими родственниками птиц следует считать триасовых псевдозухий (из надотряда <a href="/wiki/%D0%A2%D0%B5%D0%BA%D0%BE%D0%B4%D0%BE%D0%BD%D1%82%D1%8B" title="Текодонты">текодонтов</a>). Текодонтная гипотеза была общепринятой в середине XX столетия и поддерживается сегодня рядом специалистов. Однако вплоть до настоящего времени среди псевдозухий не удалось обнаружить форм, филогенетически близких к археоптериксу или другим ранним птицам.
</p>
<dl><dt>Гипотеза Уокера (архозавроморфная)</dt></dl>
<p>Алик Уокер в 1970-х годах обосновывал родство археоптерикса и всех птиц с триасовыми архозаврами из подотряда сфенозухий, родственных современным <a href="/wiki/%D0%9A%D1%80%D0%BE%D0%BA%D0%BE%D0%B4%D0%B8%D0%BB%D1%8B" title="Крокодилы">крокодилам</a><sup id="cite_ref-13" class="reference"><a href="#cite_note-13">&#91;13&#93;</a></sup>. Аргументы Уокера неоднократно критиковались, и сам автор впоследствии признал гипотезу несостоятельной.
</p><p>До недавнего времени обе гипотезы постепенно теряли сторонников. Однако после открытия четырёхкрылых оперённых животных (<a href="/wiki/%D0%9C%D0%B8%D0%BA%D1%80%D0%BE%D1%80%D0%B0%D0%BF%D1%82%D0%BE%D1%80" title="Микрораптор">микрораптора</a> и некоторых других мелких динозавров), предсказанных ещё <a href="/w/index.php?title=%D0%91%D0%B8%D0%B1,_%D0%A3%D0%B8%D0%BB%D1%8C%D1%8F%D0%BC&amp;action=edit&amp;redlink=1" class="new" title="Биб, Уильям (страница отсутствует)">Уильямом Бибом</a> в 1915 году под именем «тетраптерикс» (<i>Tetrapteryx</i>)<sup id="cite_ref-14" class="reference"><a href="#cite_note-14">&#91;14&#93;</a></sup>, группа гипотез «с деревьев вниз» вновь переживает расцвет. Что любопытно, подтвердить гипотезу Биба в 1926 году пытался Хейльман<sup id="cite_ref-15" class="reference"><a href="#cite_note-15">&#91;15&#93;</a></sup>, исследовавший ноги птенцов <a href="/wiki/%D0%93%D0%BE%D1%80%D0%BB%D0%B8%D1%86%D1%8B" title="Горлицы">горлиц</a> и <a href="/wiki/%D0%AF%D0%BA%D0%B0%D0%BD%D0%BE%D0%B2%D1%8B%D0%B5" title="Якановые">яканы</a>, однако его исследования были тогда восприняты с большим скепсисом.
</p><p>Некоторые авторы приходили к выводу о принадлежности к древним птицам <a href="/wiki/%D0%A1%D0%BA%D0%B0%D0%BD%D1%81%D0%BE%D1%80%D0%B8%D0%BE%D0%BF%D1%82%D0%B5%D1%80%D0%B8%D0%B3%D0%B8%D0%B4%D1%8B" title="Скансориоптеригиды">скансориоптеригид</a>. У <i>Scansoriopteryx</i> обнаружили характерные для птиц анатомические особенности: удлинённые передние конечности, кости запястий в форме полумесяца, лапы, приспособленные для сидения на ветках. Перья, расположенные на передних и задних конечностях, свидетельствуют о том, что он мог планировать в воздухе, перелетая между деревьями, а это, в свою очередь, означает, что полет возник не «снизу вверх», когда наземные динозавры-тероподы начали летать, а «сверху вниз», когда архозавры нарастили перья, позволившие им короткое время держаться в воздухе. Отнесение <i>Scansoriopteryx</i> к птицам вновь вызвало сомнения в происхождении птиц от динозавров<sup id="cite_ref-16" class="reference"><a href="#cite_note-16">&#91;16&#93;</a></sup>. Но дальнейшие исследования (например, Lefèvre <i>et al.</i>, 2017) это мнение не подтвердили<sup id="cite_ref-Lefèvre_2017_17-0" class="reference"><a href="#cite_note-Lefèvre_2017-17">&#91;17&#93;</a></sup>.
</p>
<h2><span id=".D0.94.D1.80.D1.83.D0.B3.D0.B8.D0.B5_.D0.B3.D0.B8.D0.BF.D0.BE.D1.82.D0.B5.D0.B7.D1.8B"></span><span class="mw-headline" id="Другие_гипотезы">Другие гипотезы</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Редактировать раздел «Другие гипотезы»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;action=edit&amp;section=4" title="Редактировать раздел «Другие гипотезы»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>После открытия <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%BE%D0%B0%D0%B2%D0%B8%D1%81" title="Протоавис">протоависа</a> Санкаром Чаттерджи (<a href="https://en.wikipedia.org/wiki/Sankar_Chatterjee" class="extiw" title="en:Sankar Chatterjee">en:Sankar Chatterjee</a>) в 1984 году появился ряд новых гипотез, ныне обычно не воспринимаемых всерьёз:
</p>
<ul><li>По мнению самого Чаттерджи, древнейшей птицей является <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%BE%D0%B0%D0%B2%D0%B8%D1%81" title="Протоавис">протоавис</a>, а не <a href="/wiki/%D0%90%D1%80%D1%85%D0%B5%D0%BE%D0%BF%D1%82%D0%B5%D1%80%D0%B8%D0%BA%D1%81" title="Археоптерикс">археоптерикс</a>. При этом ближайшими родственниками птиц являются триасовые <a href="/wiki/%D0%A6%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D0%B7%D0%B0%D0%B2%D1%80" title="Цератозавр">цератозавры</a>&#160;— группа тероподных динозавров, обладавшая рядом существенных особенностей, характерных для птиц (близкое к гетероцильному строение <a href="/wiki/%D0%A8%D0%B5%D0%B9%D0%BD%D1%8B%D0%B5_%D0%BF%D0%BE%D0%B7%D0%B2%D0%BE%D0%BD%D0%BA%D0%B8" title="Шейные позвонки">шейных позвонков</a>, сплав тазовых костей и др.). Эти особенности скелета <a href="/wiki/%D0%A6%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D1%8B" title="Цератозавры">цератозавров</a> традиционно расцениваются как результат конвергенции. При этом полёт появился в результате планирования дромерозаврид с деревьев.</li>
<li>Гипотеза дифилетического происхождения птиц, разрабатываемая <a href="/wiki/%D0%9A%D1%83%D1%80%D0%BE%D1%87%D0%BA%D0%B8%D0%BD,_%D0%95%D0%B2%D0%B3%D0%B5%D0%BD%D0%B8%D0%B9_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Курочкин, Евгений Николаевич">Е.&#160;Н.&#160;Курочкиным</a>, частично основывается на изучении <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%BE%D0%B0%D0%B2%D0%B8%D1%81" title="Протоавис">протоависа</a>. Согласно этой гипотезе, <a href="/wiki/%D0%92%D0%B5%D0%B5%D1%80%D0%BE%D1%85%D0%B2%D0%BE%D1%81%D1%82%D1%8B%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B" title="Веерохвостые птицы">веерохвостые птицы</a> произошли от некоторой группы юрских <a href="/wiki/%D0%90%D1%80%D1%85%D0%BE%D0%B7%D0%B0%D0%B2%D1%80" class="mw-redirect" title="Архозавр">архозавров</a>, близким родственником которых является протоавис. <a href="/wiki/%D0%AF%D1%89%D0%B5%D1%80%D0%BE%D1%85%D0%B2%D0%BE%D1%81%D1%82%D1%8B%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B" title="Ящерохвостые птицы">Ящерохвостые птицы</a> (например, <a href="/wiki/%D0%90%D1%80%D1%85%D0%B5%D0%BE%D0%BF%D1%82%D0%B5%D1%80%D0%B8%D0%BA%D1%81" title="Археоптерикс">археоптерикс</a> и <a href="/wiki/%D0%AD%D0%BD%D0%B0%D0%BD%D1%86%D0%B8%D0%BE%D1%80%D0%BD%D0%B8%D1%81" class="mw-redirect" title="Энанциорнис">энанциорнисы</a>) являются потомками <a href="/wiki/%D0%A2%D0%B5%D1%80%D0%BE%D0%BF%D0%BE%D0%B4%D1%8B" title="Тероподы">тероподов</a>. При этом предки птицы обитали не на настоящих деревьях, а в кустарниках, на которые они запрыгивали, используя силу ног, а спрыгивали, замедляя прыжок быстрыми махами оперённых передних конечностей. Ключевым в этой гипотезе является отсутствие стадии планирующего полёта, которая неоднократно критиковалась сторонниками гипотезы «с земли верх», согласно мнению которых, планирующий полёт самодостаточен и не способен перейти в машущий. Тем не менее, согласно Курочкину, планирующий полёт всё же имел место, но относился не к настоящим птицам, а к тупиковой ветви развития, представленной археоптериксом. Другим ключевым моментом в гипотезе Курочкина является то, что, в отличие от авторов классических гипотез «с деревьев вниз», он считает, что передние конечности при лазаньи играли вспомогательную роль и не обхватывали ствол дерева, а цеплялись за мелкие соседние ветви, помогая балансировать. Основная роль при лазаньи в гипотезе Курочкина отводится сильным задним конечностям. Это является вторым ответом критикам гипотезы «с деревьев вниз», по мнению которых, лазанье по деревьям с обхватом передними конечностями ствола дерева несовместимо с машущим полётом.</li>
<li>Гипотеза Гарнера «атакующий проавис» («prouncing proavis»)&#160;— выдвинута в 1999 году, отвергает как идею «с земли вверх», так и «с деревьев вниз». Согласно этой гипотезе, полёт появился в результате прыжков на добычу из засады с возвышения, а появление перьев служило повышению точности прыжка. В подтверждение гипотезы Гарнер приводит кладистическую линию <i>Sinosauropteryx-Protarchaeopteryx-Caudipteryx-Archaeopteryx-Iberomesornis-Enantiornithes-Ornithurae</i>, годом ранее построенную Джи.</li>
<li>Гипотеза Петерса&#160;— выдвинута в 2002 году как развитие идеи Гарнера. В качестве одного из доказательств своей гипотезы Петерс раскритиковал как классическую гипотезу «с деревьев вниз», так и классическую «с земли вверх». Так, например, по его мнению, приспособление передних конечностей к лазанью противоречит приспособлению к полёту.</li>
<li>Гипотеза Лакасы-Руиса (Lacasa-Ruiz, 1993) и Лопарёва (1996)&#160;— изначально выдвинута Лакасой-Руисом в 1993 году, а затем доработана в 1996 году С.&#160;А.&#160;Лопарёвым, который в качестве доказательства привёл сравнительный анализ строения таза птиц, а также <a href="/wiki/%D0%9E%D0%BD%D1%82%D0%BE%D0%B3%D0%B5%D0%BD%D0%B5%D0%B7" title="Онтогенез">онтогенез</a> современных птиц. Согласно гипотезе, предки птиц обитали на заросшем мелководье, и перья на передних конечностях изначально служили для создания тени с целью привлечения рыбы (так охотятся некоторые болотные птицы). При спасении от хищников махи передними конечностями облегчали бег по заросшему болотистому мелководью, что постепенно перешло в использование <a href="/wiki/%D0%AD%D0%BA%D1%80%D0%B0%D0%BD%D0%BD%D1%8B%D0%B9_%D1%8D%D1%84%D1%84%D0%B5%D0%BA%D1%82" title="Экранный эффект">экранного эффекта</a>, затем в транзитный полёт (cruising flight).</li>
<li>Гипотеза <a href="/wiki/%D0%A1%D0%B0%D0%B2%D0%B5%D0%BB%D1%8C%D0%B5%D0%B2,_%D0%A1%D0%B5%D1%80%D0%B3%D0%B5%D0%B9_%D0%92%D1%8F%D1%87%D0%B5%D1%81%D0%BB%D0%B0%D0%B2%D0%BE%D0%B2%D0%B8%D1%87" title="Савельев, Сергей Вячеславович">Савельева</a>&#160;— выдвинута в 2005. Согласно этой гипотезе, птицы изначально обитали на мелководье, а затем научились нырять, используя передние конечности подобно <a href="/wiki/%D0%9E%D0%BB%D1%8F%D0%BF%D0%BA%D0%B0_(%D0%BF%D1%82%D0%B8%D1%86%D0%B0)" class="mw-redirect" title="Оляпка (птица)">оляпке</a> и <a href="/wiki/%D0%9F%D0%B8%D0%BD%D0%B3%D0%B2%D0%B8%D0%BD" class="mw-redirect" title="Пингвин">пингвину</a>, что явилось <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%B0%D0%B4%D0%B0%D0%BF%D1%82%D0%B0%D1%86%D0%B8%D1%8F" title="Преадаптация">преадаптацией</a> к полёту<sup id="cite_ref-Saveliev_2018_18-0" class="reference"><a href="#cite_note-Saveliev_2018-18">&#91;18&#93;</a></sup><sup id="cite_ref-Kurochkin_2008_8-3" class="reference"><a href="#cite_note-Kurochkin_2008-8">&#91;8&#93;</a></sup>.</li></ul>
<h2><span id=".D0.9D.D0.B0.D0.B9.D0.B4.D0.B5.D0.BD.D0.BD.D1.8B.D0.B5_.D0.B8.D1.81.D0.BA.D0.BE.D0.BF.D0.B0.D0.B5.D0.BC.D1.8B.D0.B5"></span><span class="mw-headline" id="Найденные_ископаемые">Найденные ископаемые</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Редактировать раздел «Найденные ископаемые»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9F%D1%80%D0%BE%D0%B8%D1%81%D1%85%D0%BE%D0%B6%D0%B4%D0%B5%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86&amp;action=edit&amp;section=5" title="Редактировать раздел «Найденные ископаемые»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Эволюционное происхождение перьев можно проследить, начиная с <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%BD%D0%BE%D1%82%D0%B0%D0%B2%D1%80" title="Карнотавр">карнотавра</a>, покрытого многочисленными бугорками<sup id="cite_ref-autogenerated1_19-0" class="reference"><a href="#cite_note-autogenerated1-19">&#91;19&#93;</a></sup>, похожими на <span class="dabhide"><a href="/wiki/%D0%A4%D0%BE%D0%BB%D0%BB%D0%B8%D0%BA%D1%83%D0%BB" class="mw-disambig" title="Фолликул">фолликулы</a></span>. Следующий этап виден у <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%BE%D0%B7%D0%B0%D0%B2%D1%80%D0%BE%D0%BF%D1%82%D0%B5%D1%80%D0%B8%D0%BA%D1%81" title="Синозавроптерикс">синозавроптерикса</a> и <a href="/wiki/%D0%94%D0%B8%D0%BB%D1%83%D0%BD" title="Дилун">дилуна</a>, покрытых волокнистым пухом<sup id="cite_ref-autogenerated1_19-1" class="reference"><a href="#cite_note-autogenerated1-19">&#91;19&#93;</a></sup>. И, наконец, у <a href="/wiki/%D0%9A%D0%B0%D1%83%D0%B4%D0%B8%D0%BF%D1%82%D0%B5%D1%80%D0%B8%D0%BA%D1%81" title="Каудиптерикс">каудиптерикса</a>, <a href="/wiki/%D0%9F%D1%80%D0%BE%D1%82%D0%B0%D1%80%D1%85%D0%B5%D0%BE%D0%BF%D1%82%D0%B5%D1%80%D0%B8%D0%BA%D1%81" title="Протархеоптерикс">протархеоптерикса</a>, <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%BE%D1%80%D0%BD%D0%B8%D1%82%D0%BE%D0%B7%D0%B0%D0%B2%D1%80" title="Синорнитозавр">синорнитозавра</a> и <a href="/wiki/%D0%9C%D0%B8%D0%BA%D1%80%D0%BE%D1%80%D0%B0%D0%BF%D1%82%D0%BE%D1%80" title="Микрораптор">микрораптора</a> можно наблюдать настоящие перья<sup id="cite_ref-autogenerated1_19-2" class="reference"><a href="#cite_note-autogenerated1-19">&#91;19&#93;</a></sup>.
</p>
