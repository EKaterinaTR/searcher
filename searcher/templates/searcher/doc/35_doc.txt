<p><b>«Расскажи́те, пти́цы»</b>&#160;— <a href="/wiki/%D0%A1%D0%B8%D0%BD%D0%B3%D0%BB" title="Сингл">сингл</a> <a href="/wiki/%D0%9F%D1%83%D0%B3%D0%B0%D1%87%D1%91%D0%B2%D0%B0,_%D0%90%D0%BB%D0%BB%D0%B0_%D0%91%D0%BE%D1%80%D0%B8%D1%81%D0%BE%D0%B2%D0%BD%D0%B0" title="Пугачёва, Алла Борисовна">Аллы Пугачёвой</a>, выпущенный в <a href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" class="mw-redirect" title="СССР">СССР</a> <a href="/wiki/%D0%9C%D0%B5%D0%BB%D0%BE%D0%B4%D0%B8%D1%8F_(%D1%84%D0%B8%D1%80%D0%BC%D0%B0)" title="Мелодия (фирма)">фирмой «Мелодия»</a> в <a href="/wiki/1985_%D0%B3%D0%BE%D0%B4" title="1985 год">1985 году</a> с двумя песнями композитора <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2,_%D0%98%D0%B3%D0%BE%D1%80%D1%8C_%D0%AE%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Николаев, Игорь Юрьевич">Игоря Николаева</a>; этот <a href="/wiki/%D0%9C%D1%83%D0%B7%D1%8B%D0%BA%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9_%D1%80%D0%B5%D0%BB%D0%B8%D0%B7" title="Музыкальный релиз">релиз</a> стал для него дебютным.
</p><p>Премьера обеих песен&#160;— «Расскажите, птицы» и «Айсберг»&#160;— состоялась в декабре <a href="/wiki/1983_%D0%B3%D0%BE%D0%B4" title="1983 год">1983&#160;года</a> в концертной телепрограмме «<a href="/w/index.php?title=%D0%9D%D0%BE%D0%B2%D0%BE%D0%B3%D0%BE%D0%B4%D0%BD%D0%B8%D0%B9_%D0%B0%D1%82%D1%82%D1%80%D0%B0%D0%BA%D1%86%D0%B8%D0%BE%D0%BD_(%D1%82%D0%B5%D0%BB%D0%B5%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0)&amp;action=edit&amp;redlink=1" class="new" title="Новогодний аттракцион (телепрограмма) (страница отсутствует)">Новогодний аттракцион</a>». Тогда же состоялась студийная запись песен, однако эта запись увидела свет лишь в начале <a href="/wiki/1985_%D0%B3%D0%BE%D0%B4" title="1985 год">1985&#160;года</a>. Обе композиции пользовались большим успехом и вошли в концертный репертуар Аллы Пугачёвой середины <a href="/wiki/1980-%D0%B5" class="mw-redirect" title="1980-е">1980-х</a> годов. Сотрудничество певицы с Игорем Николаевым, начавшееся с этих песен, стало чрезвычайно плодотворным и продолжалось в течение двадцати лет; одним из результатов совместной работы стала запись в <a href="/wiki/1986_%D0%B3%D0%BE%D0%B4" title="1986 год">1986&#160;году</a> альбома «<a href="/wiki/%D0%A1%D1%87%D0%B0%D1%81%D1%82%D1%8C%D1%8F_%D0%B2_%D0%BB%D0%B8%D1%87%D0%BD%D0%BE%D0%B9_%D0%B6%D0%B8%D0%B7%D0%BD%D0%B8!" class="mw-redirect" title="Счастья в личной жизни!">Счастья в личной жизни!</a>», полностью состоящего из песен композитора.
</p><p>Песня «Айсберг» вошла в альбом «<a href="/wiki/%D0%90%D1%85,_%D0%BA%D0%B0%D0%BA_%D1%85%D0%BE%D1%87%D0%B5%D1%82%D1%81%D1%8F_%D0%B6%D0%B8%D1%82%D1%8C" title="Ах, как хочется жить">Ах, как хочется жить</a>» (1985); «Расскажите, птицы» в <a href="/wiki/%D0%A1%D0%BE%D1%8E%D0%B7_%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%B8%D1%85_%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D0%B8%D1%81%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D1%85_%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA" title="Союз Советских Социалистических Республик">СССР</a> была выпущена лишь в магнитофонной версии этого альбома и ни в одну <a href="/wiki/LP" class="mw-redirect" title="LP">долгоиграющую пластинку</a> певицы не попала, но была выпущена на её сборнике «<a href="/w/index.php?title=Alla_Pugatjova._Sovet_Superstar&amp;action=edit&amp;redlink=1" class="new" title="Alla Pugatjova. Sovet Superstar (страница отсутствует)">Alla Pugatjova. Sovet Superstar</a>», выпущенном в <a href="/wiki/1984_%D0%B3%D0%BE%D0%B4_%D0%B2_%D0%BC%D1%83%D0%B7%D1%8B%D0%BA%D0%B5" title="1984 год в музыке">1984&#160;году</a> в <a href="/wiki/%D0%A8%D0%B2%D0%B5%D1%86%D0%B8%D1%8F" title="Швеция">Швеции</a>. А в России она была включена в состав одного из альбомов Пугачёвой лишь при издании её «<a href="/wiki/%D0%9A%D0%BE%D0%BB%D0%BB%D0%B5%D0%BA%D1%86%D0%B8%D1%8F_(%D0%B1%D0%BE%D0%BA%D1%81-%D1%81%D0%B5%D1%82)" title="Коллекция (бокс-сет)">Коллекции</a>» в 1996 году.
</p><p><a href="/wiki/%D0%90%27%D0%A1%D1%82%D1%83%D0%B4%D0%B8%D0%BE" class="mw-redirect" title="А&#39;Студио">Группа «А’Студио»</a> в <a href="/wiki/1997_%D0%B3%D0%BE%D0%B4" title="1997 год">1997 году</a> записала <a href="/wiki/%D0%9A%D0%B0%D0%B2%D0%B5%D1%80-%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%8F" title="Кавер-версия">кавер-версию</a> песни «Расскажите, птицы» для <a href="/wiki/%D0%A2%D1%80%D0%B8%D0%B1%D1%8C%D1%8E%D1%82-%D0%B0%D0%BB%D1%8C%D0%B1%D0%BE%D0%BC" title="Трибьют-альбом">трибьют-альбома</a> «Сюрприз от Аллы Пугачёвой» и концертной программы «Сюрприз для Аллы», посвящённой 48-летию со Дня Рождения певицы. В концерте и на альбоме её исполнял бывший солист группы <a href="/wiki/%D0%A8%D1%83%D0%BA%D0%B5%D0%BD%D0%BE%D0%B2,_%D0%91%D0%B0%D1%82%D1%8B%D1%80%D1%85%D0%B0%D0%BD_%D0%9A%D0%B0%D0%BC%D0%B0%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Шукенов, Батырхан Камалович">Батырхан Шукенов</a>. На концерте, посвящённом 60-летию Аллы Борисовны, эту песню исполнил <a href="/wiki/%D0%91%D0%B8%D0%BB%D0%B0%D0%BD,_%D0%94%D0%B8%D0%BC%D0%B0_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Билан, Дима Николаевич">Дима Билан</a>. В <a href="/wiki/2010_%D0%B3%D0%BE%D0%B4" title="2010 год">2010 году</a> <a href="/wiki/%D0%92%D0%B0%D0%B4%D0%B8%D0%BC_%D0%A1%D0%B0%D0%BC%D0%BE%D0%B9%D0%BB%D0%BE%D0%B2" class="mw-redirect" title="Вадим Самойлов">Вадим Самойлов</a> и <a href="/wiki/%D0%9C%D0%B0%D1%88%D0%B0_%D0%9C%D0%B0%D0%BA%D0%B0%D1%80%D0%BE%D0%B2%D0%B0" class="mw-redirect" title="Маша Макарова">Маша Макарова</a> записали кавер для концерта «Песни для Аллы». Так же песня с недавних пор входит в концертный репертуар её автора <a href="/wiki/%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B5%D0%B2,_%D0%98%D0%B3%D0%BE%D1%80%D1%8C_%D0%AE%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Николаев, Игорь Юрьевич">Игоря Николаева</a>.
</p>
