<p><a href="/wiki/%D0%93%D0%B5%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D0%B8" title="География Армении">Географическое расположение</a>, <a href="/wiki/%D0%93%D0%B5%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D0%B8" title="Геология Армении">геологические структуры</a>, разнообразие
<a href="/wiki/%D0%9F%D0%BE%D1%87%D0%B2%D1%8B_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D0%B8" title="Почвы Армении">почв</a>, <a href="/wiki/%D0%9A%D0%BB%D0%B8%D0%BC%D0%B0%D1%82_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D0%B8" title="Климат Армении">климата</a> и <a href="/wiki/%D0%9B%D0%B0%D0%BD%D0%B4%D1%88%D0%B0%D1%84%D1%82%D0%BD%D1%8B%D0%B5_%D0%BF%D0%BE%D1%8F%D1%81%D0%B0_%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D0%B8" title="Ландшафтные пояса Армении">растительного покрова</a> изначально явились важным условием
формирования птичьего мира и его разнообразия. <a href="/wiki/%D0%90%D1%80%D0%BC%D1%8F%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BD%D0%B0%D0%B3%D0%BE%D1%80%D1%8C%D0%B5" title="Армянское нагорье">Армянское нагорье</a> сообщается с одной
стороны с <a href="/wiki/%D0%9E%D1%80%D0%BD%D0%B8%D1%82%D0%BE%D1%84%D0%B0%D1%83%D0%BD%D0%B0" title="Орнитофауна">оринтофаунами</a> западно-европейских, с другой стороны&#160;— переднеазиатских и
<a href="/wiki/%D0%A1%D1%80%D0%B5%D0%B4%D0%B8%D0%B7%D0%B5%D0%BC%D0%BD%D0%BE%D0%B5_%D0%BC%D0%BE%D1%80%D0%B5" title="Средиземное море">средиземноморских</a> стран. Это обстоятельство в значительной степени обуславливает
разнообразие <a href="/wiki/%D0%9E%D1%80%D0%BD%D0%B8%D1%82%D0%BE%D1%84%D0%B0%D1%83%D0%BD%D0%B0" title="Орнитофауна">орнитофауны</a>, которая составляет свыше 60&#160;% <a href="/wiki/%D0%9F%D0%BE%D0%B7%D0%B2%D0%BE%D0%BD%D0%BE%D1%87%D0%BD%D1%8B%D0%B5" title="Позвоночные">позвоночных</a> животных и
представлена 349 <a href="/wiki/%D0%91%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B2%D0%B8%D0%B4" title="Биологический вид">видами</a>.
</p><p>Количество видов птиц, обитающих на территории <a href="/wiki/%D0%90%D1%80%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" title="Армения">Армении</a>, составляет 1/3 количества видов птиц, обитающих на территории <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Европа">Европы</a>.

<p>Птицы-объекты охоты представлены следующими видами: <a href="/wiki/%D0%9A%D0%B0%D0%B2%D0%BA%D0%B0%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%82%D0%B5%D1%82%D0%B5%D1%80%D0%B5%D0%B2" title="Кавказский тетерев">кавказский тетерев</a>, <a href="/wiki/%D0%A3%D0%BB%D0%B0%D1%80" class="mw-redirect" title="Улар">улар</a>, бурая куропатка, <a href="/wiki/%D0%A1%D0%B8%D0%B7%D1%8B%D0%B9_%D0%B3%D0%BE%D0%BB%D1%83%D0%B1%D1%8C" title="Сизый голубь">сизый голубь</a>, <a href="/wiki/%D0%90%D0%B7%D0%B8%D0%B0%D1%82%D1%81%D0%BA%D0%B8%D0%B9_%D0%BA%D0%B5%D0%BA%D0%BB%D0%B8%D0%BA" title="Азиатский кеклик">каменная куропатка</a>, лысуха черная, бекас чернушка, большой водолаз, малый водолаз, <a href="/wiki/%D0%9A%D1%80%D1%8F%D0%BA%D0%B2%D0%B0" title="Кряква">кряква</a>, <a href="/wiki/%D0%A1%D0%B5%D1%80%D0%B0%D1%8F_%D1%83%D1%82%D0%BA%D0%B0" title="Серая утка">серая утка</a>, <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%B5%D0%BF%D0%B5%D0%BB" class="mw-redirect" title="Перепел">перепел</a>, <a href="/wiki/%D0%9B%D0%B5%D1%81%D0%BD%D0%BE%D0%B9_%D0%B3%D0%BE%D0%BB%D1%83%D0%B1%D1%8C" class="mw-redirect" title="Лесной голубь">лесной голубь</a>, <a href="/wiki/%D0%93%D0%BE%D1%80%D0%BB%D0%B8%D1%86%D1%8B" title="Горлицы">горлица</a>, <a href="/wiki/%D0%A1%D0%B5%D1%80%D1%8B%D0%B9_%D0%B6%D1%83%D1%80%D0%B0%D0%B2%D0%BB%D1%8C" title="Серый журавль">серый журавль</a>, прелестный стрепет, <a href="/wiki/%D0%91%D0%B5%D0%BA%D0%B0%D1%81" title="Бекас">бекас</a>, <a href="/wiki/%D0%A1%D0%B5%D1%80%D1%8B%D0%B9_%D0%B3%D1%83%D1%81%D1%8C" title="Серый гусь">серый гусь</a>, пёстрая утка и&#160;т.&#160;д..
