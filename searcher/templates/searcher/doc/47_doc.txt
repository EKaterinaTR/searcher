<p><b>Домашние птицы</b> (<a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">англ.</a>&#160;<span lang="en" style="font-style:italic;">poultry</span>)&#160;— <a href="/wiki/%D0%9E%D0%B4%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5" title="Одомашнивание">одомашненные</a> <a href="/wiki/%D0%9F%D1%82%D0%B8%D1%86%D1%8B" title="Птицы">птицы</a>, разводимые людьми в <a href="/wiki/%D0%A1%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%BE%D0%B5_%D1%85%D0%BE%D0%B7%D1%8F%D0%B9%D1%81%D1%82%D0%B2%D0%BE" title="Сельское хозяйство">сельскохозяйственных</a> целях&#160;— для получения <a href="/wiki/%D0%9C%D1%8F%D1%81%D0%BE" title="Мясо">мяса</a>, <a href="/wiki/%D0%AF%D0%B9%D1%86%D0%BE_(%D0%BF%D0%B8%D1%89%D0%B5%D0%B2%D0%BE%D0%B9_%D0%BF%D1%80%D0%BE%D0%B4%D1%83%D0%BA%D1%82)" title="Яйцо (пищевой продукт)">яиц</a>, <a href="/wiki/%D0%9F%D0%B5%D1%80%D0%BE" title="Перо">перьев</a>, а также в <a href="/wiki/%D0%94%D0%B5%D0%BA%D0%BE%D1%80%D0%B0%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B5_%D0%BF%D0%BE%D1%80%D0%BE%D0%B4%D1%8B_%D0%BA%D1%83%D1%80" title="Декоративные породы кур">декоративных</a> целях. Эти птицы, как правило, относятся к <a href="/wiki/%D0%9A%D0%BB%D0%B0%D0%B4%D0%B0" title="Клада">кладе</a> (<a href="/wiki/%D0%9D%D0%B0%D0%B4%D0%BE%D1%82%D1%80%D1%8F%D0%B4" class="mw-redirect" title="Надотряд">надотряду</a>) <a href="/wiki/Galloanserae" class="mw-redirect" title="Galloanserae">Galloanserae</a> и отряду <a href="/wiki/Galliformes" class="mw-redirect" title="Galliformes">Galliformes</a> (<a href="/wiki/%D0%9A%D1%83%D1%80%D0%BE%D0%BE%D0%B1%D1%80%D0%B0%D0%B7%D0%BD%D1%8B%D0%B5" title="Курообразные">курообразные</a>); последний включает <a href="/wiki/%D0%9A%D1%83%D1%80%D0%B8%D1%86%D0%B0" title="Курица">кур</a>, <a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%BE%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D0%BF%D0%B5%D0%BB" title="Немой перепел">перепелов</a>, <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D1%8F%D1%8F_%D0%B8%D0%BD%D0%B4%D0%B5%D0%B9%D0%BA%D0%B0" title="Домашняя индейка">индеек</a> и <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D1%8F%D1%8F_%D1%86%D0%B5%D1%81%D0%B0%D1%80%D0%BA%D0%B0" title="Домашняя цесарка">цесарок</a>, а также <a href="/wiki/%D0%9E%D0%B1%D1%8B%D0%BA%D0%BD%D0%BE%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BF%D0%B0%D0%B2%D0%BB%D0%B8%D0%BD" title="Обыкновенный павлин">павлинов</a> и <a href="/wiki/%D0%9E%D0%B1%D1%8B%D0%BA%D0%BD%D0%BE%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D1%84%D0%B0%D0%B7%D0%B0%D0%BD" title="Обыкновенный фазан">фазанов</a>. В категорию одомашненных птиц также включают <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D1%8F%D1%8F_%D1%83%D1%82%D0%BA%D0%B0" title="Домашняя утка">уток</a>, <a href="/wiki/%D0%9C%D1%83%D1%81%D0%BA%D1%83%D1%81%D0%BD%D0%B0%D1%8F_%D1%83%D1%82%D0%BA%D0%B0" title="Мускусная утка">мускусных уток</a>, <a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B9_%D0%B3%D1%83%D1%81%D1%8C" title="Домашний гусь">гусей</a>, <a href="/wiki/%D0%93%D0%BE%D0%BB%D1%83%D0%B1%D0%B8" title="Голуби">голубей</a>, <a href="/wiki/%D0%A1%D1%82%D1%80%D0%B0%D1%83%D1%81" class="mw-redirect" title="Страус">страусов</a>, но не включают диких птиц, на которых <a href="/wiki/%D0%9E%D1%85%D0%BE%D1%82%D0%B0" title="Охота">охотятся</a> из-за <a href="/wiki/%D0%A1%D0%BF%D0%BE%D1%80%D1%82" title="Спорт">спортивного</a> интереса или для употребления в <a href="/wiki/%D0%9F%D0%B8%D1%89%D0%B0" title="Пища">пищу</a> и которых называют <a href="/wiki/%D0%94%D0%B8%D1%87%D1%8C" title="Дичь">дичью</a>. <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D0%B9%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Английский язык">Английское</a> слово <span lang="en" style="font-style:italic;">«poultry»</span> происходит от <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D1%83%D0%B7%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Французский язык">фр.</a>&#160;<span lang="fr" style="font-style:italic;">«poule»</span>&#160;— «курица», которое само по себе происходит от <a href="/wiki/%D0%9B%D0%B0%D1%82%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Латинский язык">лат.</a>&#160;<span lang="la" style="font-style:italic;">«pullum»</span>, что означает «<span class="dabhide"><a href="/wiki/%D0%A6%D1%8B%D0%BF%D0%BB%D1%91%D0%BD%D0%BE%D0%BA" class="mw-disambig" title="Цыплёнок">цыплёнок</a></span>».
</p>

<h2><span id=".D0.98.D1.81.D1.82.D0.BE.D1.80.D0.B8.D1.8F"></span><span class="mw-headline" id="История">История</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Редактировать раздел «История»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=1" title="Редактировать раздел «История»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Приручение диких птиц началось несколько тысяч лет назад. Первоначально это могло быть результатом того, что люди собирали яйца в <a href="/wiki/%D0%94%D0%B8%D0%BA%D0%B0%D1%8F_%D0%BF%D1%80%D0%B8%D1%80%D0%BE%D0%B4%D0%B0" title="Дикая природа">дикой природе</a> и выводили из них птиц, но позже это привело к постоянному содержанию птиц в неволе. В настоящее время, в результате <a href="/wiki/%D0%AD%D0%B2%D0%BE%D0%BB%D1%8E%D1%86%D0%B8%D1%8F_%D0%BF%D1%82%D0%B8%D1%86" class="mw-redirect" title="Эволюция птиц">эволюции видов птиц</a> и <a href="/wiki/%D0%98%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%BE%D1%82%D0%B1%D0%BE%D1%80" title="Искусственный отбор">искусственного отбора</a> на протяжении веков с целью получения новых <a href="/wiki/%D0%9F%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_(%D0%B6%D0%B8%D0%B2%D0%BE%D1%82%D0%BD%D0%BE%D0%B2%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE)" title="Порода (животноводство)">пород</a>, имеющих более быстрый <span class="dabhide"><a href="/wiki/%D0%A0%D0%BE%D1%81%D1%82" class="mw-disambig" title="Рост">рост</a></span>, увеличенную <a href="/wiki/%D0%AF%D0%B9%D1%86%D0%B5%D0%BD%D0%BE%D1%81%D0%BA%D0%BE%D1%81%D1%82%D1%8C" title="Яйценоскость">яйценоскость</a>, лучшее качество <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B5%D0%BD%D0%B8%D0%B5_(%D0%B1%D0%B8%D0%BE%D0%BB%D0%BE%D0%B3%D0%B8%D1%8F)" title="Оперение (биология)">оперения</a>, современные породы часто сильно отличаются от своих диких <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%B4%D0%BE%D0%BA" title="Предок">предков</a>.
</p><p>Для удовлетворения <a href="/wiki/%D0%A0%D1%8B%D0%BD%D0%BE%D0%BA" title="Рынок">рыночного</a> <a href="/wiki/%D0%A1%D0%BF%D1%80%D0%BE%D1%81" title="Спрос">спроса</a> большинство птиц, доступных на рынке сегодня, выращиваются на <a href="/wiki/%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D1%8F" class="mw-redirect" title="Предприятия">предприятиях</a> интенсивного промышленного <a href="/wiki/%D0%9F%D1%82%D0%B8%D1%86%D0%B5%D0%B2%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE" title="Птицеводство">птицеводства</a> (<span class="dabhide"><a href="/wiki/%D0%9F%D1%82%D0%B8%D1%86%D0%B5%D1%84%D0%B0%D0%B1%D1%80%D0%B8%D0%BA%D0%B0_(%D0%B7%D0%BD%D0%B0%D1%87%D0%B5%D0%BD%D0%B8%D1%8F)" class="mw-disambig" title="Птицефабрика (значения)">птицефабриках</a></span>) с использованием коммерческих <a href="/wiki/%D0%9A%D1%80%D0%BE%D1%81%D1%81%D1%8B_%D0%B4%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B5%D0%B9_%D0%BF%D1%82%D0%B8%D1%86%D1%8B" title="Кроссы домашней птицы">кроссов</a> и <a href="/wiki/%D0%A7%D0%B8%D1%81%D1%82%D0%B0%D1%8F_%D0%BB%D0%B8%D0%BD%D0%B8%D1%8F" title="Чистая линия">линий</a>, но небольшие количества домашней, в том числе <a href="/wiki/%D0%9F%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_(%D0%B6%D0%B8%D0%B2%D0%BE%D1%82%D0%BD%D0%BE%D0%B2%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE)" title="Порода (животноводство)">породной</a>, птицы содержатся в мелких и <a href="/wiki/%D0%A7%D0%B0%D1%81%D1%82%D0%BD%D0%BE%D0%B5_%D0%BF%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5" class="mw-redirect" title="Частное предприятие">частных</a> хозяйствах. Важное значение имеют оценка, сохранение и использование <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%BE%D1%84%D0%BE%D0%BD%D0%B4" title="Генофонд">генофонда</a> <a href="/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D1%80%D0%B0%D0%B7%D0%BD%D0%BE%D0%BE%D0%B1%D1%80%D0%B0%D0%B7%D0%B8%D0%B5" title="Генетическое разнообразие">разнообразных</a> <a href="/wiki/%D0%90%D0%B1%D0%BE%D1%80%D0%B8%D0%B3%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BF%D0%BE%D1%80%D0%BE%D0%B4%D0%B0" title="Аборигенная порода">местных</a>, <span data-interwiki-lang="en" data-interwiki-article="Rare breed (agriculture)"><a href="/w/index.php?title=%D0%A0%D0%B5%D0%B4%D0%BA%D0%B0%D1%8F_%D0%BF%D0%BE%D1%80%D0%BE%D0%B4%D0%B0_(%D0%B6%D0%B8%D0%B2%D0%BE%D1%82%D0%BD%D0%BE%D0%B2%D0%BE%D0%B4%D1%81%D1%82%D0%B2%D0%BE)&amp;action=edit&amp;redlink=1" class="new" title="Редкая порода (животноводство) (страница отсутствует)">редких</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a href="https://en.wikipedia.org/wiki/Rare_breed_(agriculture)" class="extiw" title="en:Rare breed (agriculture)"><span title="Rare breed (agriculture) — версия статьи «Редкая порода (животноводство)» на английском языке">[en]</span></a></sup></span> и исчезающих пород птицы<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup><sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup>.

<h3><span id=".D0.9A.D1.83.D1.80.D1.8B"></span><span class="mw-headline" id="Куры">Куры</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Редактировать раздел «Куры»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=3" title="Редактировать раздел «Куры»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основные статьи: <b><a href="/wiki/%D0%9A%D1%83%D1%80%D0%B8%D1%86%D0%B0" title="Курица">Курица</a></b> и <b><a href="/wiki/%D0%9F%D0%BE%D1%80%D0%BE%D0%B4%D1%8B_%D0%BA%D1%83%D1%80" title="Породы кур">Породы кур</a></b></div>
<p>Одомашнены в период между 3500 и 2000&#160;гг. до&#160;н.&#160;э.<sup id="cite_ref-O2000_3-1" class="reference"><a href="#cite_note-O2000-3">&#91;3&#93;</a></sup><sup id="cite_ref-O3200_4-1" class="reference"><a href="#cite_note-O3200-4">&#91;4&#93;</a></sup><sup id="cite_ref-O3500_5-1" class="reference"><a href="#cite_note-O3500-5">&#91;5&#93;</a></sup>. Распространение по миру произошло между 1500 и 500&#160;гг. до&#160;н.&#160;э.<sup id="cite_ref-Sil_8-0" class="reference"><a href="#cite_note-Sil-8">&#91;8&#93;</a></sup>. Птицы средних размеров, нелетающие, всеядные, вес колеблется от 1,5&#160;кг до 5&#160;кг. (петухи крупнее, чем куры), яйценоскость от 150 до 250 штук в год в зависимости от породы. Породы делятся на яичные, яично-мясные, мясные, декоративные и бойцовые<sup id="cite_ref-9" class="reference"><a href="#cite_note-9">&#91;9&#93;</a></sup><sup id="cite_ref-10" class="reference"><a href="#cite_note-10">&#91;10&#93;</a></sup>.
</p>
<h3><span id=".D0.9F.D0.B5.D1.80.D0.B5.D0.BF.D0.B5.D0.BB.D0.B0"></span><span class="mw-headline" id="Перепела">Перепела</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Редактировать раздел «Перепела»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=4" title="Редактировать раздел «Перепела»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основные статьи: <b><a href="/wiki/%D0%9D%D0%B5%D0%BC%D0%BE%D0%B9_%D0%BF%D0%B5%D1%80%D0%B5%D0%BF%D0%B5%D0%BB" title="Немой перепел">Немой перепел</a></b> и <b><a href="/wiki/%D0%9F%D0%BE%D1%80%D0%BE%D0%B4%D1%8B_%D0%BF%D0%B5%D1%80%D0%B5%D0%BF%D0%B5%D0%BB%D0%BE%D0%B2" title="Породы перепелов">Породы перепелов</a></b></div>
<p>Одомашнены в конце XIX&#160;— начале XX века в Японии. Птицы маленькие, хорошо летающие, всеядные, вес колеблется от 150 до 300 граммов (самочки крупнее самцов), яйценоскость до 300 штук в год в зависимости от породы. Породы делятся на яйценосские, мясные и общепользовательские<sup id="cite_ref-11" class="reference"><a href="#cite_note-11">&#91;11&#93;</a></sup><sup id="cite_ref-YP_12-0" class="reference"><a href="#cite_note-YP-12">&#91;12&#93;</a></sup>.
</p>
<h3><span id=".D0.98.D0.BD.D0.B4.D0.B5.D0.B9.D0.BA.D0.B8"></span><span class="mw-headline" id="Индейки">Индейки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=5" class="mw-editsection-visualeditor" title="Редактировать раздел «Индейки»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=5" title="Редактировать раздел «Индейки»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основные статьи: <b><a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D1%8F%D1%8F_%D0%B8%D0%BD%D0%B4%D0%B5%D0%B9%D0%BA%D0%B0" title="Домашняя индейка">Домашняя индейка</a></b> и <b><a href="/wiki/%D0%9F%D0%BE%D1%80%D0%BE%D0%B4%D1%8B_%D0%B8%D0%BD%D0%B4%D0%B5%D0%B5%D0%BA" title="Породы индеек">Породы индеек</a></b></div>
<p>Одомашнивание в <a href="/wiki/%D0%9D%D0%BE%D0%B2%D1%8B%D0%B9_%D0%A1%D0%B2%D0%B5%D1%82" title="Новый Свет">Новом Свете</a> задолго до открытия Америки европейцами (точные даты неизвестны)<sup id="cite_ref-13" class="reference"><a href="#cite_note-13">&#91;13&#93;</a></sup>. Доставлены в Европу испанцами в 1519 году, оттуда через несколько лет распространились по всей Европе. Птицы крупные, вторые по величине домашние птицы после страусов. Вес самцов достигает 35&#160;кг, самки гораздо мельче (в три раза). Нелетающие<sup id="cite_ref-YP_12-1" class="reference"><a href="#cite_note-YP-12">&#91;12&#93;</a></sup>.
</p>
<h3><span id=".D0.A3.D1.82.D0.BA.D0.B8"></span><span class="mw-headline" id="Утки">Утки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=6" class="mw-editsection-visualeditor" title="Редактировать раздел «Утки»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=6" title="Редактировать раздел «Утки»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основные статьи: <b><a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D1%8F%D1%8F_%D1%83%D1%82%D0%BA%D0%B0" title="Домашняя утка">Домашняя утка</a></b> и <b><a href="/wiki/%D0%9F%D0%BE%D1%80%D0%BE%D0%B4%D1%8B_%D1%83%D1%82%D0%BE%D0%BA" title="Породы уток">Породы уток</a></b></div>
<p>Домашние утки известны в Месопотамии с конца 4&#160;— начала 3 тыс. до&#160;н.&#160;э.<sup id="cite_ref-Me_6-1" class="reference"><a href="#cite_note-Me-6">&#91;6&#93;</a></sup>. Птицы средних размеров, плохо летающие, водоплавающие, хорошо ныряют. Вес селезней 3,6—4,2&#160;кг, уток 3,4—3,9&#160;кг, яйценоскость в зависимости от породы 100—340 яиц в год
<sup id="cite_ref-14" class="reference"><a href="#cite_note-14">&#91;14&#93;</a></sup><sup id="cite_ref-15" class="reference"><a href="#cite_note-15">&#91;15&#93;</a></sup><sup id="cite_ref-k0150001_16-0" class="reference"><a href="#cite_note-k0150001-16">&#91;16&#93;</a></sup><sup id="cite_ref-k0170002_17-0" class="reference"><a href="#cite_note-k0170002-17">&#91;17&#93;</a></sup>.
</p>
<h3><span id=".D0.93.D1.83.D1.81.D0.B8"></span><span class="mw-headline" id="Гуси">Гуси</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=7" class="mw-editsection-visualeditor" title="Редактировать раздел «Гуси»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=7" title="Редактировать раздел «Гуси»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основные статьи: <b><a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B9_%D0%B3%D1%83%D1%81%D1%8C" title="Домашний гусь">Домашний гусь</a></b> и <b><a href="/wiki/%D0%9F%D0%BE%D1%80%D0%BE%D0%B4%D1%8B_%D0%B4%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D1%85_%D0%B3%D1%83%D1%81%D0%B5%D0%B9" title="Породы домашних гусей">Породы домашних гусей</a></b></div>
<p>Время одомашнивания точно неизвестно, предположительно в период между 1500 и 1000&#160;гг. до&#160;н.&#160;э. Хотя относятся к водоплавающим, плавают гораздо хуже уток, предпочитают большую часть времени проводить на суше. 
</p><p>Масса взрослых гусаков 5—6&#160;кг, наибольшая до 12&#160;кг, гусынь&#160;— 4—5&#160;кг, наибольшая до 10&#160;кг. Яйценоскость составляет 30—40 яиц, у некоторых пород&#160;— до 100 яиц<sup id="cite_ref-18" class="reference"><a href="#cite_note-18">&#91;18&#93;</a></sup>.
</p>
<h3><span id=".D0.A6.D0.B5.D1.81.D0.B0.D1.80.D0.BA.D0.B8"></span><span class="mw-headline" id="Цесарки">Цесарки</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=8" class="mw-editsection-visualeditor" title="Редактировать раздел «Цесарки»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=8" title="Редактировать раздел «Цесарки»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="hatnote navigation-not-searchable">Основная статья: <b><a href="/wiki/%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D1%8F%D1%8F_%D1%86%D0%B5%D1%81%D0%B0%D1%80%D0%BA%D0%B0" title="Домашняя цесарка">Домашняя цесарка</a></b></div>
<p>В античные времена домашняя цесарка попала из Африки в Древнюю Грецию (где упоминается как домашняя птица уже в V в. до н. э.) и Древний Рим. Португальские путешественники завезли цесарку повторно в Европу из Западной Африки в XV и XVI веках, после чего стала популярна на птичьих дворах всего мира как домашняя птица.
</p><p>Живая масса взрослых самцов — 1,5-1,6 кг, самок — 1,6-1,7 кг, цесарята 70-дневного возраста — около 850 грамм. Яйценоскость сезонная, за 5-6 месяцев в среднем на несушку получают 80-90 яиц массой 45 грамм.
</p>

<h2><span id=".D0.A6.D0.B5.D0.BD.D0.BD.D0.BE.D1.81.D1.82.D1.8C_.D0.BC.D1.8F.D1.81.D0.B0_.D0.BF.D1.82.D0.B8.D1.86.D1.8B_.D0.BA.D0.B0.D0.BA_.D0.BF.D1.80.D0.BE.D0.B4.D1.83.D0.BA.D1.82.D0.B0_.D0.BF.D0.B8.D1.82.D0.B0.D0.BD.D0.B8.D1.8F"></span><span class="mw-headline" id="Ценность_мяса_птицы_как_продукта_питания">Ценность мяса птицы как продукта питания</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=12" class="mw-editsection-visualeditor" title="Редактировать раздел «Ценность мяса птицы как продукта питания»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=12" title="Редактировать раздел «Ценность мяса птицы как продукта питания»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Мясо домашней птицы находится на втором месте (около 30&#160;%) по наиболее употребляемому типу мяса в мире после <a href="/wiki/%D0%A1%D0%B2%D0%B8%D0%BD%D0%B8%D0%BD%D0%B0" title="Свинина">свинины</a> (38&#160;%). Мясо и яйца домашней птицы содержат белок высокого качества. Это сопровождается низким содержанием жиров, которые содержат благоприятную смесь жирных кислот<sup id="cite_ref-FAOhealth_21-0" class="reference"><a href="#cite_note-FAOhealth-21">&#91;21&#93;</a></sup>. Куриное мясо содержит примерно в два-три раза больше полиненасыщенных жиров, чем большинство видов красного мяса, если измерять по весу<sup id="cite_ref-22" class="reference"><a href="#cite_note-22">&#91;22&#93;</a></sup>. Тем не менее, для куриной грудки без кожи это количество намного ниже. Порция запеченной куриной грудки весом 100 г содержит 4 г жира и 31 г белка по сравнению с 10 г жира и 27 г белка на ту же порцию жареного стейка<sup id="cite_ref-23" class="reference"><a href="#cite_note-23">&#91;23&#93;</a></sup><sup id="cite_ref-24" class="reference"><a href="#cite_note-24">&#91;24&#93;</a></sup>.
</p>

<h3><span id=".D0.91.D0.B0.D0.BA.D1.82.D0.B5.D1.80.D0.B8.D0.B0.D0.BB.D1.8C.D0.BD.D0.BE.D0.B5_.D0.B7.D0.B0.D1.80.D0.B0.D0.B6.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Бактериальное_заражение">Бактериальное заражение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=14" class="mw-editsection-visualeditor" title="Редактировать раздел «Бактериальное заражение»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=14" title="Редактировать раздел «Бактериальное заражение»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Исследование, проведенное <span data-interwiki-lang="en" data-interwiki-article="Translational Genomics Research Institute"><a href="/w/index.php?title=%D0%9D%D0%B0%D1%83%D1%87%D0%BD%D0%BE-%D0%B8%D1%81%D1%81%D0%BB%D0%B5%D0%B4%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D1%82%D1%80%D0%B0%D0%BD%D1%81%D0%BB%D1%8F%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%BE%D0%B9_%D0%B3%D0%B5%D0%BD%D0%BE%D0%BC%D0%B8%D0%BA%D0%B8&amp;action=edit&amp;redlink=1" class="new" title="Научно-исследовательский институт трансляционной геномики (страница отсутствует)">Научно-исследовательским институтом трансляционной геномики</a></span><span class="noprint" style="font-style:normal; font-weight:normal;"><sup><a href="https://en.wikipedia.org/wiki/Translational_Genomics_Research_Institute" class="extiw" title="en:Translational Genomics Research Institute"><span title="Translational Genomics Research Institute — версия статьи «Научно-исследовательский институт трансляционной геномики» на английском языке">[en]</span></a></sup></span> в Аризоне в 2011 году показало, что 47&#160;% мяса и птицы, продаваемых в продуктовых магазинах Соединённых Штатов, были заражены <a href="/wiki/%D0%97%D0%BE%D0%BB%D0%BE%D1%82%D0%B8%D1%81%D1%82%D1%8B%D0%B9_%D1%81%D1%82%D0%B0%D1%84%D0%B8%D0%BB%D0%BE%D0%BA%D0%BE%D0%BA%D0%BA" title="Золотистый стафилококк">золотистым стафилококком</a>, а 52&#160;% этих бактерий показали устойчивость по крайней мере к трём группам антибиотиков. Исследователи заявили, что при варке продукта эти бактерии можно уничтожить, но риск перезаражения вследствие неправильного обращения с сырыми продуктами после этого все ещё остается<sup id="cite_ref-25" class="reference"><a href="#cite_note-25">&#91;25&#93;</a></sup>. Кроме того, для потребителей мяса птицы и яиц существует также некоторый риск заражения бактериями, такими как <a href="/wiki/%D0%A1%D0%B0%D0%BB%D1%8C%D0%BC%D0%BE%D0%BD%D0%B5%D0%BB%D0%BB%D0%B0" class="mw-redirect" title="Сальмонелла">сальмонелла</a> и <a href="/wiki/%D0%9A%D0%B0%D0%BC%D0%BF%D0%B8%D0%BB%D0%BE%D0%B1%D0%B0%D0%BA%D1%82%D0%B5%D1%80" class="mw-redirect" title="Кампилобактер">кампилобактер</a>. Продукты, получаемые на птицефабриках, могут загрязняться этими бактериями во время обработки или хранения<sup id="cite_ref-FAOhealth_21-1" class="reference"><a href="#cite_note-FAOhealth-21">&#91;21&#93;</a></sup>.
</p>
<h3><span id=".D0.9F.D1.82.D0.B8.D1.87.D0.B8.D0.B9_.D0.B3.D1.80.D0.B8.D0.BF.D0.BF"></span><span class="mw-headline" id="Птичий_грипп">Птичий грипп</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=15" class="mw-editsection-visualeditor" title="Редактировать раздел «Птичий грипп»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=15" title="Редактировать раздел «Птичий грипп»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Птичий грипп&#160;— это заболевание птиц, вызываемое птичьим вирусом гриппа А, который обычно не передается людям; однако люди, контактирующие с живой птицей, подвергаются наибольшему риску заражения этим вирусом, и это вызывает особую обеспокоенность в таких регионах, как Юго-Восточная Азия, где это заболевание является эндемичным для <a href="/wiki/%D0%9F%D0%BE%D0%BF%D1%83%D0%BB%D1%8F%D1%86%D0%B8%D1%8F" title="Популяция">популяции</a> диких птиц и может привести к заражению домашних птиц. Вирус, по-видимому, может мутировать, чтобы стать высоко вирулентным и заразным для людей и вызвать пандемию гриппа<sup id="cite_ref-26" class="reference"><a href="#cite_note-26">&#91;26&#93;</a></sup>.
</p>
<h3><span id=".D0.A1.D0.BE.D0.B7.D0.B4.D0.B0.D0.BD.D0.B8.D0.B5_.D0.B2.D0.B0.D0.BA.D1.86.D0.B8.D0.BD"></span><span class="mw-headline" id="Создание_вакцин">Создание вакцин</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;veaction=edit&amp;section=16" class="mw-editsection-visualeditor" title="Редактировать раздел «Создание вакцин»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%94%D0%BE%D0%BC%D0%B0%D1%88%D0%BD%D0%B8%D0%B5_%D0%BF%D1%82%D0%B8%D1%86%D1%8B&amp;action=edit&amp;section=16" title="Редактировать раздел «Создание вакцин»">править код</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Бактерии можно выращивать в лаборатории на питательных культуральных средах, но вирусам нужны живые клетки, чтобы размножаться. Многие вакцины от инфекционных заболеваний можно выращивать в оплодотворенных куриных яйцах. Миллионы яиц используются каждый год для покрытия ежегодных потребностей в вакцинах против гриппа. Это сложный процесс, который занимает около шести месяцев после определения, какие штаммы вируса следует включить в новую вакцину. Проблема с использованием яиц для этой цели заключается в том, что люди с аллергией на яйца не могут быть иммунизированы, но этот недостаток может быть преодолён, когда станут доступны новые методы клеточного культивирования вместо использования культур на базе яиц<sup id="cite_ref-27" class="reference"><a href="#cite_note-27">&#91;27&#93;</a></sup>. Клеточная культура также будет полезна при эпидемиях, когда, возможно, будет трудно получить достаточно большое количество подходящих стерильных оплодотворенных яиц<sup id="cite_ref-28" class="reference"><a href="#cite_note-28">&#91;28&#93;</a></sup><sup id="cite_ref-29" class="reference"><a href="#cite_note-29">&#91;29&#93;</a></sup>.
</p>

<p>Существуют также <a href="/wiki/%D0%9D%D0%B0%D1%81%D0%BB%D0%B5%D0%B4%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%B7%D0%B0%D0%B1%D0%BE%D0%BB%D0%B5%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F" title="Наследственные заболевания">наследуемые (врождённые) аномалии</a> птицы, в связи с чем актуальным является их <span class="dabhide"><a href="/wiki/%D0%A1%D0%BA%D1%80%D0%B8%D0%BD%D0%B8%D0%BD%D0%B3_(%D0%B7%D0%BD%D0%B0%D1%87%D0%B5%D0%BD%D0%B8%D1%8F)" class="mw-disambig" title="Скрининг (значения)">скрининг</a></span> в популяциях домашних птиц<sup id="cite_ref-30" class="reference"><a href="#cite_note-30">&#91;30&#93;</a></sup><sup id="cite_ref-31" class="reference"><a href="#cite_note-31">&#91;31&#93;</a></sup>.
</p>
